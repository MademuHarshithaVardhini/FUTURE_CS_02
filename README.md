# Phishing Email Detection & Awareness

## Objective
Analyze phishing email samples and identify phishing indicators to improve user awareness.

## Tools Used
- Kali Linux
- Google Message Header Analyzer
- MXToolbox
- Whois and Dig commands

## Methodology
1. Collected phishing email samples
2. Extracted email headers
3. Analyzed email routing
4. Identified phishing indicators
5. Classified risk level
6. Created awareness guidelines

## Outcome
Created a phishing awareness report that demonstrates how to identify phishing emails and prevent cyber attacks.
